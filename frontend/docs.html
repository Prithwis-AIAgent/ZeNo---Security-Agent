Documentation

Everything you need to set up, configure, and run SecureAgent locally.

1. Installation

SecureAgent is built with Python 3.11+. Follow these steps to get the MCP server running locally.

# 1. Clone the repository
git clone https://github.com/your-org/secureagent.git
cd secureagent

# 2. Create and activate a virtual environment
python -m venv .venv

# Windows:
.venv\Scripts\activate
# macOS/Linux:
source .venv/bin/activate

# 3. Install dependencies
pip install -r requirements.txt

2. Configuration

SecureAgent uses a .env file for configuration. Start by copying the example file:

cp .env.example .env
Variable Required Description
OPENAI_API_KEY Yes* Your OpenAI API key for LLM enrichment.
GEMINI_API_KEY Yes* Google Gemini API key (alternative to OpenAI).
LLM_PROVIDER No Set to openai or gemini.
LLM_MODEL No Model name. We recommend gpt-4.1-mini for best cost/performance.
GITHUB_TOKEN No Increases GitHub API rate limit from 60 to 5000 requests/hr.

* If no API key is provided, the server falls back to Pattern-Only mode automatically.

3. Usage Guide

Start the FastAPI server using Uvicorn:

uvicorn mcp_server.main:app --reload --host 0.0.0.0 --port 8000

Once running, the interactive Swagger UI is available at http://localhost:8000/docs. You can execute tools directly from the browser.

4. API Reference

POST /tools/audit_repo

Audits a public GitHub repository. Scans source code and dependency files.

curl -X POST "http://localhost:8000/tools/audit_repo" \
     -H "Content-Type: application/json" \
     -d '{"github_url": "https://github.com/owner/repo"}'

POST /tools/audit_code

Audits a raw code snippet. Useful for checking code before committing.

curl -X POST "http://localhost:8000/tools/audit_code" \
     -H "Content-Type: application/json" \
     -d '{
         "code": "import pickle\ndata = pickle.loads(user_input)",
         "filename": "app.py"
     }'

Response Format

Both endpoints return the same JSON structure containing the markdown report and severity counts.

{
  "report": "# SecureAgent Security Report\n\n## 🔴 Critical\n...",
  "summary": {
    "critical": 1,
    "high": 0,
    "medium": 0,
    "low": 0,
    "total": 1
  },
  "target": "inline:app.py"
}
